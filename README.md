# 🚀 My DSA Practice Repository

Welcome to my **Data Structures and Algorithms (DSA) Practice Repository**! 
This repository contains various **DSA implementations**, **algorithmic solutions**, and **LeetCode problem solutions** written in **Java**. 
It serves as a reference for my **coding practice** and preparation for **coding interviews**.

---

## 📂 **Repository Structure**
The repository is organized into different categories of **DSA topics**:

📌 **Arrays** - Implementation of array-based problems and algorithms.  
📌 **Dynamic Programming** - Optimized DP solutions for common problems.  
📌 **Graphs** - Graph representations, BFS, DFS, shortest paths, etc.  
📌 **Heaps** - MinHeap, MaxHeap, and Priority Queue implementations.  
📌 **Linked Lists** - Singly & Doubly Linked List implementations.  
📌 **Queues** - Queue and Priority Queue implementations.  
📌 **Stacks** - Stack implementations, including dynamic stacks.  
📌 **Sorting & Searching** - Sorting algorithms like QuickSort, MergeSort.  
📌 **Strings** - String manipulation algorithms and pattern matching.  
📌 **Trees** - Binary Trees, Binary Search Trees, Segment Trees, etc.  
📌 **LeetCode** - Solutions to various LeetCode problems.
